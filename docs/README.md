# 기능

## 유저
- [ ] 이름과 배팅 금액을 갖는다.
- [ ] 카드를 받는다.
- [ ] 카드 숫자를 계산한다.
### 딜러
- [ ] 처음 받은 2장의 합계가 16이하면 반드시 1장의 카드를 추가로 받고, 아니면 못 받는다.
### 플레이어

## 카드들
- [ ] 가능한 카드의 합을 모두 반환한다.

---

## 카드팩
- [ ] 카드들을 생성한다.
- [ ] 카드를 랜덤하게 반환한다.

---

## 카드 팩토리
- [ ] 한 모양의 카드들을 생성한다.

## 카드
- [ ] 카드가 사용되면 체크된다.
- [ ] 모양과 이름을 가진다.
### 하나의 숫자를 갖는 카드
- [ ] 숫자를 반환한다.
### 여러개의 숫자를 갖는 카드
- [ ] 숫자들을 반환한다.
